## Add components from CLI

There are two ways to add components to your project using the CLI.

1. Using the components URL. (Note you will need to specify the style, e.g `https://somaui.dev/new-york/button`)

2. Using the `REGISTRY_URL` environment variable.

### Using the components URL (Recommended)

<PackageManagerTabs>
  <PackageManagerContent value="npm">
    ```bash
    npx shadcn@latest add https://somaui.dev/[[STYLE]]/button
    ```
  </PackageManagerContent>
  <PackageManagerContent value="pnpm">
    ```bash
    pnpm dlx shadcn@latest add https://somaui.dev/[[STYLE]]/button
    ```

  </PackageManagerContent>
  <PackageManagerContent value="bun">
    ```bash
    bunx --bun shadcn@latest add https://somaui.dev/[[STYLE]]/button
    ```
  </PackageManagerContent>
  <PackageManagerContent value="yarn">
    ```bash 
    npx shadcn@latest add https://somaui.dev/[[STYLE]]/button
    ```
  </PackageManagerContent>
</PackageManagerTabs>
